## Add a new plant to a location

1. Open Plants.
2. Select a species.
3. Create an individual plant record.
4. Set status, label, notes, and planting date.
5. Assign location.

Expected result: plant appears in both plant lists and location context views.

## Move a plant from tray to bed

1. Open Seedlings/Trays.
2. Open tray cell assignment and identify source plant.
3. Update plant location to target bed.
4. Add a journal note with transplant date.

Expected result: tray cell is cleared or reassigned; plant history remains intact.

## Create and resolve an issue

1. Open Issues.
2. Create issue with linked plant/location and priority.
3. Add comments as actions are taken.
4. Transition issue status to `closed` after validation.

Expected result: complete issue lifecycle with traceable actions.

## Set moisture alerts for a bed

1. Add or open bed moisture sensor.
2. Set min and max limits.
3. Confirm alert enabled flag.
4. Check readings for threshold violations.

Expected result: threshold breaches create visible attention points in
monitoring workflows.
