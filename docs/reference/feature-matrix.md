| Area | Primary Entities | Typical Operations |
| --- | --- | --- |
| Environments | Environment, Location | Create, update, map placement, hierarchy |
| Plants | Species, Plant, Plant Group | Catalog lookup, plant lifecycle tracking |
| Seedlings | Seedling Tray, Tray Cell, Observation | Assign cells, monitor growth, transplant |
| Indoor | Indoor Environment, Indoor Reading | Log climate/reservoir data, set targets |
| Schedules | Schedule, Schedule Run | Recurrence setup, run history |
| Sensors | Sensor, Sensor Reading, Sensor Limit | Attach, threshold monitoring, anomaly checks |
| Issues | Issue, Label, Comment | Detect, triage, resolve |
| Journal | Journal Entry | Chronological notes and context |
| Reports | Harvest, Season, analytics outputs | Summaries and trend review |
| Integrations | Integration Config, Sync Run, map settings | Provider configuration and sync |
| REST API | All core entities | HTTP read/write access for plugins and 3rd-party tools |
| Backup/Import | Export payload, import replay | Snapshot, transfer, restore |
